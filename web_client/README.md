# React_starter
just a react web-client starter

### Based on 
- react
- webpack
- yarn
- babel
- (with some ESlint, for the beauty of the code)

### Clone the depo
```bash 
git clone https://github.com/tashikomaaa/React_starter.git
```

### Install all the dependencies 
```bash 
cd React_starter && npm install -save
```

### Start the project 
```bash
yarn start
```
